# Textile sub-repo template

Use this folder as the base for each textile repo.

## Create a new portfolio repo
Example repo names:
- ikat-blue
- blockprint-earth
- woven-neutral

Upload these files to the repo root and enable GitHub Pages.

## Replace these parts
- Change the page title and heading in `index.html`
- Replace preview images in `/images`
- Edit `works.json`
- Add `cover.jpg` in the repo root for the card shown on the main homepage

## Important protection practice
Do not upload:
- full repeat files
- production PDFs
- high-resolution exports
- exact scale charts

Use only:
- cropped previews
- low-resolution images
- visible watermarking inside the image itself if possible

## Limits
This code reduces casual copying only. It does not stop screenshots or determined extraction.
